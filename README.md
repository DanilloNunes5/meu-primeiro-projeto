APLICAÇÃO WEB - FRONTEND BACKEND E BANCO DE DADOS

PROJETO INTEGRADOR 

Aplicaçãp WEB será um e-commerce de rótulos de vinho, onde será possível ter acesso a informações sobre o tipo de uva e o tipo de fermentação
de cada rótulo.

Telas e Funcionalidades:
1- Home - Aqui será apresentado como funciona o modelo de négocio.
2- Tela de Consultas de rotulos presentes dentro do negócio.
3- Tela de criação/login de usuário
4- A partir do login terá acesso ao carrinho do e-commerce onde será implementado o CRUD.

OBS: Poderá ter diversas informações sobre apreciação de vinhos - como um modelo de educação de paladar e relacional com a mercadoria.
